# Financial-EDA-Project
Developed an end-to-end Python-based data analysis and machine learning project on financial loan data, performing detailed EDA, data cleaning, and feature engineering to uncover key risk patterns. Identified that high loan amounts and lower credit grades contributed to ~25–35% higher default probability, enabling data-driven decision making for risk segmentation. Built and evaluated predictive models achieving ~85–90% accuracy using metrics like precision, recall, and F1-score. Proposed actionable solutions such as stricter loan approval policies for high-risk customers and targeted financial strategies to reduce defaults and improve portfolio performance.
